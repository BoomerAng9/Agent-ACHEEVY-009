## 6. Responsive Behavior

### Mobile (< 768px)
- VoiceOrb fills 60% of viewport width
- ConversationStarters stack vertically, full-width cards
- RehearsalStage stacks (your side on top, companion below)
- Sidebar collapses to bottom sheet
- Athlete/Deal cards are single column
- CircleLobby participants scroll horizontally

### Tablet (768â€“1439px)
- VoiceOrb at 80px, centered
- ConversationStarters in 2-column grid
- RehearsalStage side-by-side
- Athlete/Deal cards in 2-column grid
- CircleLobby shows all participants

### Desktop (1440px+)
- Full layout as wireframed above
- 4-column grids for cards
- Split views for sessions
- Persistent sidebar navigation
